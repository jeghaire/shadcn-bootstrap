This is my Next.js, tailwindcss, Shadcn/ui startup template

## Getting Started

First, run

```bash
npm install
# or
yarn install
# or
pnpm install
# or
bun install
```

then run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
